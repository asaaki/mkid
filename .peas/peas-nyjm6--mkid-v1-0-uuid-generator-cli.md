First version of mkid, a Rust-based CLI tool for generating UUIDs (v1-v8). Provides better alternative to uuidgen with support for all UUID versions, multiple output formats, and clipboard integration. Default command generates UUID v7 (lowercase, hyphenated). Includes comprehensive testing and documentation.
